# Advanced Usage & Developer Workflows

<Callout type="success">
  Pro tip: Use <b>sigyl dev</b> for rapid iteration and hot-reload during development!
</Callout>

<Accordion title="CLI Advanced Workflows">
  - <b>Custom MCP server:</b> Use <code>sigyl init</code> or <code>sigyl scan</code> to generate a new server, then customize <code>server.ts</code> and <code>sigyl.yaml</code>.
  - <b>Integration:</b> Use <code>sigyl integrate</code> to add MCP endpoints to an existing Express app.
  - <b>Dev mode:</b> Use <code>sigyl dev</code> for hot-reload development.
  - <b>Environment variables:</b> Use for local dev/testing.
</Accordion>

<Accordion title="SDK Advanced Workflows">
  - <b>Authenticated operations:</b> Pass <code>requireAuth: true</code> and an API key.
  - <b>Registering packages:</b> Use <code>registerMCP</code> with full package/tool schema.
  - <b>Direct tool invocation:</b> Use <code>connectDirect</code> or <code>invoke</code> for custom endpoints.
  - <b>TypeScript support:</b> All types are exported for type-safe development.
</Accordion>
