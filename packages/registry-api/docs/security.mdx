# Security & Best Practices

<Callout type="warning">
  Never commit API keys to version control!
</Callout>

<Accordion title="Best Practices">
  - <b>Use environment variables</b> for secrets.
  - <b>Rotate API keys</b> regularly.
  - <b>Use different keys</b> for dev and prod.
  - <b>Monitor API usage</b> via the Sigyl dashboard.
  - <b>CLI:</b> All usage is tracked and rate-limited by the registry API.
</Accordion>

<Callout type="tip">
  For extra security, restrict API key permissions and audit usage logs frequently.
</Callout>
