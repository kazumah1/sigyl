# Troubleshooting

<Callout type="info">
  Still stuck? Reach out on <a href="https://discord.gg/sigyl">Discord</a> for help from the community and maintainers.
</Callout>

<Accordion title="Package not found">
  Ensure the package exists in the registry. See <a href="https://sigyl.dev/marketplace">sigyl.dev/marketplace</a>.
</Accordion>
<Accordion title="Authentication failed">
  Check your API key and registry URL. Get a new key at <a href="https://sigyl.dev/dashboard">sigyl.dev/dashboard</a>.
</Accordion>
<Accordion title="Connection issues">
  Check your internet connection and registry URL.
</Accordion>
<Accordion title="CLI errors">
  Use <code>sigyl config show</code> to verify your config.
</Accordion>
<Accordion title="SDK errors">
  Ensure your API key and registry URL are correct.
</Accordion>
